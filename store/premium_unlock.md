# In-app product: `premium_unlock`

The one-time **Premium** purchase (managed product, non-consumable). The product id `premium_unlock`
lives in [`scripts/billing.gd`](../scripts/billing.gd); the title, description and price are entered by
hand in the Play Console (Monetize -> Products -> In-app products). Versioned here so the copy stays in
sync with the app and the [store listing](README.md).

**Title** (max 55 chars) -- 20:

```
Limpid Chess Premium
```

**Description** (max 200 chars) -- 195:

```
Unlock everything with one purchase, forever: unlimited games, puzzles and game reviews, all bots, and Face to Face on one device. No ads, no subscription. Thanks for supporting a solo indie dev!
```

Note: uses **Face to Face** (the mode's user-facing name), not "Pass & Play". The description is
English only for now; it can be localized in the Play Console like the listing if wanted.
